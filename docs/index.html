Predicting Antimicrobial Resistance using Deep Learning

A Species-Agnostic Approach Using Whole Genome Sequences

We developed a cutting-edge deep learning framework that predicts antimicrobial resistance (AMR) directly from bacterial whole genome sequences. By transforming genomic data into images using Frequency Chaos Game Representation (FCGR), our models achieve state-of-the-art accuracy across multiple bacterial species and antibiotics, offering a rapid, scalable solution to combat the global AMR crisis.

FCGR Transformation

We use Frequency Chaos Game Representation (FCGR) to transform variable-length genomic sequences into fixed-size 256×256 grayscale images based on 8-mer frequencies. This alignment-free approach preserves biological information while enabling the use of powerful computer vision architectures.

Data Processing Pipeline: FASTA sequences → 8-mer extraction → FCGR encoding → 256×256 images → ResNet18 CNN

Key Features: Homology-aware splitting (95-98% similarity threshold), masked evaluation metrics, 5-fold cross-validation with GroupKFold

Results & Performance

Comprehensive evaluation across species and resistance mechanisms

Species-Specific Models

Individual models trained for each species demonstrated robust performance, with accuracy ranging from 72% (M. tuberculosis) to 95% (Salmonella enterica).

  • Salmonella enterica: 95.0% accuracy, 96.6% F1-score
  • Staphylococcus aureus: 83.9% accuracy, 69.5% F1-score
  • Acinetobacter baumannii: 88.3% accuracy, 57.2% F1-score
  • Escherichia coli: 81.6% accuracy, 51.8% F1-score

Species-Agnostic Model

The unified model trained on three species achieved exceptional performance while maintaining generalizability across bacterial pathogens.

  • Overall Accuracy: 91.3%
  • Precision: 93.0%
  • Recall: 93.2%
  • F1-Score: 92.9%
  • AUC: 95.2%

Clinical Significance

Our models consistently outperformed traditional rule-based systems like ABRicate across all tested species and antibiotics.

  • Speed: Predictions in minutes vs. hours/days
  • Coverage: Novel resistance patterns detected
  • Scalability: Database-independent approach
  • Reliability: High precision reduces false positives

Model Architecture & Demo

Modified ResNet18 architecture trained on FCGR images

ResNet18 Architecture

We employ a modified ResNet18 architecture with custom input layer for grayscale FCGR images and adapted output layer for multi-drug resistance prediction.

Input: 256×256 grayscale FCGR image

Architecture: ResNet18 (modified first conv layer)

Output: Multi-label classification (R/S per drug)

Training: 5-fold cross-validation, masked metrics

Optimization: Adam optimizer, weighted loss for class imbalance

Scientific Paper

Read our complete research findings and methodology

Predicting Antimicrobial Resistance using Whole Genome Sequence Data: A Deep Learning Approach

Saifelden Mohamed, Mariam Ismael, Salma Hatem
Zewail City of Science and Technology, 2025

Abstract

Antimicrobial resistance (AMR) poses a significant global health threat. We present a comprehensive deep learning framework for predicting AMR directly from bacterial whole genome sequences using Frequency Chaos Game Representation (FCGR). Our approach achieved accuracies ranging from 72% to 95% across 66 antibiotics and 6 bacterial species, with a species-agnostic model demonstrating 91% accuracy. This work contributes to rapid AMR screening in clinical settings.

Key Pipeline Features

  • Homology-aware splitting: 95-98% similarity threshold using Sourmash (k=31) to prevent data leakage
  • Balanced dataset curation: R/S ratio between 0.3-3.0, minimum 1000 samples per class
  • Masked metrics: Custom evaluation metrics that ignore NULL values in multi-label predictions
  • 5-fold cross-validation: Robust performance estimation with GroupKFold
  • Class weighting: Positive weights applied during training to handle imbalanced classes
